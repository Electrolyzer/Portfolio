NanoPStat

Professional iOS app for electrochemical analysis and data visualization

Swift SwiftUI CoreBluetooth CoreData CloudKit

Project Overview

Solo Developer
7.9MB App Size
iOS 17+ Compatibility

NanoPStat is a professional iOS application that connects to Analytical Instrument Systems (AIS) electrochemical devices via Bluetooth. The app enables scientists and researchers to configure experiments, collect real-time data, and analyze results directly on their iOS devices with seamless cloud synchronization.

As the solo developer, I built this app from the ground up using modern iOS technologies including SwiftUI for the interface, CoreBluetooth for device communication, CoreData for local storage, and CloudKit for cross-device data synchronization.

App Interface

Bluetooth Device Connection

Seamless Bluetooth connectivity using CoreBluetooth framework. The app automatically discovers and connects to compatible AIS electrochemical devices including DLK-NPx, DLK-MP1, and DLK-Wellstat series.

CoreBluetooth Device Discovery Connection Management

Experiment Configuration

Intuitive experiment setup interface allowing researchers to configure voltammetric analyses and stripping analysis parameters. Settings are saved locally using CoreData for quick access to frequently used configurations.

SwiftUI Forms CoreData Parameter Validation

Real-time Data Visualization

Live data plotting with real-time updates as experiments run. The interface provides immediate feedback on data collection progress with smooth, responsive charts that update as data streams from the connected device.

Real-time Plotting Data Streaming SwiftUI Charts

Advanced Serial Sequence Configuration

Sophisticated parameter configuration for complex experimental sequences. The interface provides fine-grained control over device settings while maintaining an intuitive user experience through careful SwiftUI design.

Complex UI Parameter Management User Experience

Data Analysis & Comparison

Comprehensive data analysis tools with multi-experiment comparison, interactive zoom and pan, selective data visualization, and advanced smoothing algorithms. Results are automatically synced to iCloud for cross-device access.

CloudKit Sync Data Analysis Interactive Charts

Comprehensive Settings

Detailed options menu providing access to all app features including data export, device management, and user preferences. Clean, organized interface following iOS design guidelines for optimal user experience.

iOS Design Guidelines Settings Management Data Export

Technical Implementation

iOS Development Stack

Built entirely in Swift using SwiftUI for a modern, declarative user interface. The app leverages the latest iOS frameworks and follows Apple's design guidelines for professional-grade applications.

  • SwiftUI for responsive, declarative UI
  • Combine framework for reactive programming
  • iOS 17+ compatibility with backward support
  • Universal app supporting iPhone, iPad, and Mac

Bluetooth Communication

Implemented robust Bluetooth connectivity using CoreBluetooth framework to communicate with multiple series of electrochemical devices. Handles device discovery, connection management, and real-time data streaming.

  • Multi-device support (DLK-NPx, DLK-MP1, DLK-Wellstat)
  • Automatic device discovery and pairing
  • Real-time data streaming and parsing
  • Connection state management and error handling

Data Management

Sophisticated data architecture using CoreData for local experiment storage and CloudKit for seamless cross-device synchronization. Ensures data integrity and availability across all user devices.

  • CoreData for local experiment configurations
  • CloudKit for result synchronization
  • Automatic backup and restore functionality
  • Data export capabilities for further analysis

Advanced Data Visualization

Custom-built charting system with real-time updates, interactive controls, and advanced signal processing. Implements sophisticated smoothing algorithms for clean data presentation.

  • Real-time plotting with smooth animations
  • Interactive zoom, pan, and selection
  • Multi-experiment comparison views
  • Advanced smoothing with sliding window algorithms

Professional Features

Enterprise-grade features designed for scientific research environments, including comprehensive experiment management, data analysis tools, and professional data export options.

  • Experiment template system
  • Batch data processing capabilities
  • Professional data export formats
  • Cross-platform data sharing

Development Approach

Solo development project showcasing end-to-end iOS development skills, from initial architecture design through App Store deployment. Emphasis on clean code, user experience, and maintainable architecture.

  • MVVM architecture pattern
  • Comprehensive error handling
  • Extensive testing and validation
  • App Store optimization and deployment
